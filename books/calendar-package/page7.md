# 謝辞

今回 Flutter のパッケージ開発に焦点を合わせて、カレンダーアプリを作り切ることをテーマに設定しました。

実際の現場でもロジックと Widget に分けてアプリケーションの開発を進めることが多く、基礎固めにとって良い教材となることと考えています。

また、これまで Flutter にわずかな興味しか持たなかった方も、これをきっかけにして少しでも考察する時間にあてていただけましたら幸いです。
